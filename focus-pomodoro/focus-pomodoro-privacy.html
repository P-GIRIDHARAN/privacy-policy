Privacy Policy — Focus Pomodoro Timer

Last updated: December 9, 2025

This privacy policy explains how the Focus Pomodoro Timer browser extension handles data. The extension is intentionally minimal and privacy-focused. It stores only the information necessary for the timer to function, and all stored data stays on the user's device.

What data this extension collects

Short answer: None of your personal or browsing data is collected, transmitted, or shared.

The extension only uses local storage to persist the following timer state and preferences so the timer continues to work reliably across tabs and sessions:

  • Work duration (minutes)
  • Break duration (minutes)
  • Current timer state (time remaining, whether it is running or paused, and whether it is a work or break session)
  • Notification preference (on/off for in-page popups)
No personally identifiable information, browsing history, location, communication content, or analytics are collected by the extension.

How data is stored

All state and preferences are saved to chrome.storage.local (the browser's local storage). This data is stored on the user's device and is not uploaded to any external server.

Permissions and why they are required

The extension requests only the permissions that are necessary for its single purpose (running an on-page Pomodoro timer):

  • notifications — needed to show optional browser notifications when a work or break session starts or ends.
  • storage — used to save user preferences and timer state so sessions persist across tabs and browser restarts.
  • <all_urls> (content script match) — used to inject the floating timer widget into pages the user visits. The content script injects UI only and does not read or transmit page content.

Remote code

The extension does not load any remote JavaScript, WebAssembly, or other executable code. All code (HTML, CSS, JavaScript) is packaged with the extension and executed locally in the browser.

Third parties and data sharing

There is no sharing, selling, or transfer of user data to third parties. The extension does not include analytics, advertising, or tracking libraries.

Security

Because the extension does not transmit data externally and stores only minimal preferences locally, the attack surface is small. Users should nonetheless keep their browser and extensions up to date to receive security fixes.
