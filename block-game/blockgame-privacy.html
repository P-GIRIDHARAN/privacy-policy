Privacy Policy for Block Game Extension

Effective Date: October 26, 2023

This Privacy Policy explains how the Block Game Chrome extension handles user data.

1. Data Collection and Usage

The Block Game extension is designed for a single purpose: to provide a quick, in-browser block puzzle game. The extension does not collect, store, or transmit any Personally Identifiable Information (PII).

What Data We Collect (Non-PII):

  • User Activity/Game State: We collect data related to your progress within the game, including your Score, current Level, Lines Cleared, and Options (e.g., starting level, theme preference).

How We Use This Data:

  • All collected data is used solely for the functionality of the Block Game extension (e.g., loading your last game score, setting the difficulty based on your preference).
  • This data is stored locally within your browser's synchronized storage (using chrome.storage.sync) and is never transmitted to our servers or any external party.

2. Data Sharing and Transfer

  • We do not sell, rent, or transfer any user data (including game state data) to advertisers or any third parties.
  • The data is not used for any purposes unrelated to the extension's single purpose.

3. Data Storage and Security

All game data is managed by the Chrome browser's built-in storage mechanisms and is subject to the security policies of Google Chrome.

4. Changes to This Policy

We reserve the right to update this Privacy Policy at any time. Any changes will be posted on this page with an updated "Effective Date."

5. Contact Information

If you have any questions about this Privacy Policy, please contact the developer via the extension's listing page on the Chrome Web Store.
